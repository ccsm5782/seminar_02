# 🏦 기업 부도예측 신용평가모형 개발

미국 기업 패널 데이터(1999~2018년, 8,971개사)를 활용해 **기업신용평가 모형**을 개발하는 프로젝트입니다. 📊
김종윤(2019) 박사논문의 개인신용평가 방법론(Fine/Coarse Classing, WOE/IV, 로지스틱 회귀, 평점표)을 기업 버전으로 적용합니다. 📖
시계열 파생변수(변화율, 이동평균, 변동성 등) **400개+** 를 생성하고 NICE디앤비 공시 기준(PSI ≤ 0.15, KS ≥ 20, AR ≥ 30)으로 모형을 검증합니다. ⚙️
최종 산출물은 **AAA~D 10단계** 기업신용등급 체계와 등급별 부도율 매핑입니다. 🏅
